1. Приложение написано в виде веб приложения. Для проверки запустите файл -> rup_app.py(открытый терминал не закрывать).

2. Веб_сервер работает в отладочном режиме, для перехода на веб страницу запустите ярлык Птицы.

3. Навигация url:

Добавить птицу					- http://localhost:8000/create_bird/
Просмотреть всех птиц в приложении		- http://localhost:8000/all_bird/
Просмотреть конкретную птицу			- http://localhost:8000/view_bird/<int:pk>
Просмотреть просмотреть отмеченную птицу	- http://localhost:8000/recently_view/ 

4. Для того что бы во вкладке отмеченной птицы  появилась птица ее нужна до этого отметить в просмотре(так как отмеченная птица хранится в глобальной переменной по тех. заданию,
 т.е после запуска сервера переменная пуста,  после закрытие сервера переменная удаляется).
